## 📌 About This Repository

This repository serves as the **centralized asset storage** for the FitBuddy graduation project — a comprehensive fitness tracking mobile application built with React Native and Expo.

All visual assets in this repository were **designed and produced by Mohammad Alkhatib** using AI-assisted design tools, edited, branded, and curated to meet the visual standards of the FitBuddy application.

---

## 📁 Repository Structure

```
Fit-Buddy-Assets/
│
├── 📂 Exercises-GIFS/
│   └── Animated GIF files for each exercise (no subfolders)
│
├── 📂 Logo/
│   └── FitBuddy logo files (PNG)
│
└── 📂 mockups/
    └── Mid-fidelity mockup screens for the FitBuddy app
```

## 🎨 Asset Specifications

- **Exercise GIFs:** Animated 3D illustrations and branded with FitBuddy logo
- **Logo:** High resolution PNG, transparent background
- **Mockups:** Mid-fidelity screen designs covering core app flows

---

## 🔗 How to Use Assets in the App

Replace the URL format as follows for any GIF:

```
https://raw.githubusercontent.com/C0DEEXX/Fit-Buddy-Assets/main/Exercises-GIFS/YOUR_FILE_NAME.gif
```
